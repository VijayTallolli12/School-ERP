# Teacher User Guide

Version: 1.0.0

Revision date: 2026-07-08

## Login

Sign in using your teacher account and access the teacher dashboard.

![Screenshots Placeholder](../assets/screenshots/teacher-login.png)

## Dashboard

The teacher dashboard highlights timetable, attendance, homework, student management, marks, leave, documents, payslips, notifications, and calendar information.

## Sidebar

The teacher sidebar provides quick access to assigned academic and personal workflows.

## Daily Tasks

- Review the day timetable.
- Record or review attendance.
- Create homework assignments.
- Review student information and marks.
- Manage leave requests and documents.

## Attendance

Attendance can be updated and reviewed through the attendance area.

## Homework

Create and edit homework for assigned class sections.

## Reports

Use available teacher reports for subjects and attendance when access is enabled.

## Notifications

Review in-app notifications and operational alerts.

## AI

Use Ask ERP for school-related questions.

## Profile and Settings

Update profile details and account preferences.

## Logout

Use the logout action.

## Common Errors

- Missing permissions: confirm teacher role assignment.
- Data not visible: verify the active school context.

## FAQ

- Can teachers create homework? Yes, when the role has the corresponding permission.

## Best Practices

- Keep attendance records updated.
- Review the timetable and notifications daily.
